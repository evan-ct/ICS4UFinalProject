# Étozi

Étozi is a French verb-conjugation learning web app built for my ICS4U final project. It uses Streamlit, SQLite, bcrypt password hashing, and a local French conjugation engine through `verbecc`.

## Features

- Account creation and login
- French verb conjugation practice
- Grammar practice levels
- User dashboard with accuracy charts
- Achievements and collectible character profiles
- Class system for teachers and students
- Homework/devoir assignment tracking
- Gacha-style Rouler page using earned pages

## Tech Stack

- Python
- Streamlit
- SQLite
- bcrypt
- pandas
- Altair
- verbecc
